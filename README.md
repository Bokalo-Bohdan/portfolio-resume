##Interactive Portfolio Webpage

Modern interactive web-resume for demonstration front-end development skills.

##Getting Started

This instructions will help you to run web-resume on your PC.

##Prerequisites

All soft are linked with CDN so you don't need to install any.

##Installation

Clone repository on your local device:

##Bash

git clone https://github.com/Bokalo_Bohdan/your-repo-name.git

Go to folder with project.

Open index.html in your browser.

##Features

- GSAP Animations
- ScrollTrigger Effects
- Interactive Sidebar
- Form Integration
